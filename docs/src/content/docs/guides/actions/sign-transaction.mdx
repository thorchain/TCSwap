# Signing Transactions and Messages

SwapKit provides ways to sign transactions and messages across different blockchains. This guide shows how to use these signing capabilities with both granular and all-in-one package approaches.

## Two Approaches for Using SwapKit

You can use SwapKit in two ways:

1. **Granular approach** - Import only what you need from individual packages (recommended for frontend):
   ```ts
   import { SwapKit } from "@swapkit/core";
   import { evmWallet } from "@swapkit/wallets/evm-extensions";
   ```

2. **All-in-one approach** - Import everything from the SDK (better for backend/Node.js):
   ```ts
   import { createSwapKit } from "@swapkit/sdk";
   ```

## Signing Messages

Signing messages is useful for authentication, proving ownership of an address, or interacting with decentralized applications.

### Granular Approach

```ts twoslash
// @noErrorValidation
import { Chain } from "@swapkit/helpers";
import { SwapKit } from "@swapkit/core";
import { evmWallet } from "@swapkit/wallets/evm-extensions";

// Initialize SwapKit
const swapKit = SwapKit({
  wallets: { ...evmWallet },
});

// Connect a wallet
async function signMessageExample() {
  // Connect an Ethereum wallet
  await swapKit.connectEVMWallet([Chain.Ethereum]);

  // Message to sign
  const message = "I am proving ownership of this wallet address";

  try {
    // Sign the message
    const signature = await swapKit.signMessage({
      chain: Chain.Ethereum,
      message,
    });

    return signature;
  } catch (error) {
    throw error;
  }
}
```

### All-in-one Approach

```ts twoslash
// @noErrorValidation
import { Chain } from "@swapkit/helpers";
import { createSwapKit } from "@swapkit/sdk";

// Initialize SwapKit
const swapKit = createSwapKit();

// Connect a wallet and sign a message
async function signMessageExample() {
  // Connect an Ethereum wallet
  await swapKit.connectKeystore([Chain.Ethereum], "your secret phrase here");

  // Message to sign
  const message = "I am proving ownership of this wallet address";

  try {
    // Sign the message
    const signature = await swapKit.signMessage({
      chain: Chain.Ethereum,
      message,
    });

    return signature;
  } catch (error) {
    throw error;
  }
}
```

## Signing with Different Chains

Different blockchains have different message signing formats:

```ts twoslash
// @noErrorValidation
import { Chain } from "@swapkit/helpers";
import { SwapKit } from "@swapkit/core";
import { evmWallet } from "@swapkit/wallets/evm-extensions";
import { keystoreWallet } from "@swapkit/wallets/keystore";

// Initialize SwapKit
const swapKit = SwapKit({
  wallets: { ...evmWallet, ...keystoreWallet },
});

// Connect different wallets
async function setupWallets() {
  // Connect wallets for multiple chains
  await swapKit.connectKeystore(
    [Chain.Bitcoin, Chain.Cosmos, Chain.THORChain],
    "your secret phrase here"
  );

  // Connect EVM wallet
  await swapKit.connectEVMWallet([Chain.Ethereum]);
}

// Sign with Cosmos/THORChain
async function signCosmosMessage() {
  const message = "Sign this message to verify your THORChain address";

  try {
    const signature = await swapKit.signMessage({
      chain: Chain.THORChain,
      message,
    });

    return signature;
  } catch (error) {
    throw error;
  }
}
```

## Verifying Signed Messages

After signing a message, you or others may want to verify the signature:

```ts twoslash
// @noErrorValidation
import { Chain } from "@swapkit/helpers";
import { SwapKit } from "@swapkit/core";
import { keystoreWallet } from "@swapkit/wallets/keystore";

// Initialize SwapKit
const swapKit = SwapKit({
  wallets: { ...keystoreWallet },
});

// Verify a signed message (currently supported for THORChain)
async function verifySignature() {
  // Message that was signed
  const message = "I am proving ownership of this wallet address";

  // Address that supposedly signed the message
  const address = "thor1fkm5w5uum0j5h9g35vwqtf7angh9uvrm9mszyx";

  // The signature to verify
  const signature = "ac7bfd5b5c9f44788fb533b7f08b213a3e3856e75ecdb19c5e9138ba29c4c1681e82b7de139e0b92cd9442297ec7fdd2c14e7034a3aa417ff9756e9b591d93201";

  try {
    // Verify the signature
    const isValid = await swapKit.verifyMessage({
      chain: Chain.THORChain,
      message,
      address,
      signature,
    });

    return isValid;
  } catch (error) {
    return false;
  }
}
```

## Signing Transactions

In most cases, SwapKit handles transaction signing internally when using methods like `transfer` or `swap`. However, there are cases where you might want to create and sign a transaction separately.

### Using Toolboxes for Transaction Signing

For more control over the signing process, you can use toolboxes directly:

```ts twoslash
// @noErrorValidation
import { Chain } from "@swapkit/helpers";
import { getEvmToolbox } from "@swapkit/toolboxes/evm";
import { ethers } from "ethers";

// Create and sign an Ethereum transaction manually
async function signEthereumTransaction() {
  // Create a provider and signer
  const provider = new ethers.JsonRpcProvider("https://mainnet.infura.io/v3/your-infura-key");
  const signer = new ethers.Wallet("your-eth-private-key", provider);

  // Get the EVM toolbox
  const ethToolbox = await getEvmToolbox(Chain.Ethereum, {
    provider,
    signer,
  });

  // Create a transaction to sign
  const transaction = {
    to: "0x742d35Cc6634C0532925a3b844Bc454e4438f44e",
    value: ethers.parseEther("0.01"), // 0.01 ETH
    data: "0x", // No data for a simple transfer
    from: signer.address,
  };

  try {
    // Sign transaction (but don't send it)
    const signedTx = await ethToolbox.sendTransaction(transaction);

    // Later, you can broadcast the signed transaction
    // const txHash = await ethToolbox.broadcastTransaction(signedTx);

    return signedTx;
  } catch (error) {
    throw error;
  }
}
```

### Bitcoin Transaction Signing

For UTXO chains like Bitcoin, the process involves creating inputs and outputs:

```ts twoslash
// @noErrorValidation
import { AssetValue, Chain, FeeOption } from "@swapkit/helpers";
import { getUtxoToolbox } from "@swapkit/toolboxes/utxo";

async function signBitcoinTransaction() {
  // Get the Bitcoin toolbox
  const btcToolbox = await getUtxoToolbox(Chain.Bitcoin);

  // Create a wallet from a mnemonic phrase
  const keys = await btcToolbox.createKeysForPath({
    phrase: "your twelve word mnemonic phrase here",
    derivationPath: "m/84'/0'/0'/0/0", // BIP84 for native segwit
  });

  const fromAddress = btcToolbox.getAddressFromKeys(keys);

  // Amount to send
  const amount = AssetValue.from({
    chain: Chain.Bitcoin,
    symbol: "BTC",
    value: "0.001", // 0.001 BTC
  });

  // Recipient address
  const recipient = "bc1qar0srrr7xfkvy5l643lydnw9re59gtzzwf5mdq";

  try {
    // Build the transaction without broadcasting
    const { psbt: builder, utxos } = await btcToolbox.buildTx({
      sender: fromAddress,
      recipient,
      assetValue: amount,
      feeRate: 1.2,
      memo: "Test transaction",
    });

    // Sign the transaction
    // @ts-expect-error
    utxos.forEach((utxo, index) => builder.sign(index, keys));

    // Later, you can broadcast it
    const txHash = await btcToolbox.broadcastTx(builder.toHex());
  } catch (error) {
    throw error;
  }
}
```

## Custom Transaction Signing with Hardware Wallets

Hardware wallets like Ledger or Trezor have special requirements for signing:

```ts twoslash
// @noErrorValidation
import { AssetValue, Chain, FeeOption } from "@swapkit/helpers";
import { SwapKit } from "@swapkit/core";
import { ledgerWallet } from "@swapkit/wallets/ledger";

// Initialize SwapKit
const swapKit = SwapKit({
  wallets: { ...ledgerWallet },
});

// Connect Ledger wallet and sign a transaction
async function signWithLedger() {
  try {
    // Connect Ledger
    await swapKit.connectLedger([Chain.Ethereum]);

    const ledgerWallet = swapKit.getWallet(Chain.Ethereum);

    // Create a simple ETH transfer
    const txHash = await ledgerWallet.transfer({
      assetValue: AssetValue.from({ chain: Chain.Ethereum, value: "0.01" }),
      feeOptionKey: FeeOption.Fast,
      from: ledgerWallet.address,
      memo: "Ledger transfer",
      recipient: "0x742d35Cc6634C0532925a3b844Bc454e4438f44e",
    });

    return txHash;
  } catch (error) {
    throw error;
  }
}
```

## Using Low-level Toolbox Signing Methods

For complete control over the signing process, you can use toolbox-specific methods:

```ts twoslash
// @noErrorValidation
import { Chain } from "@swapkit/helpers";
import { getCosmosToolbox } from "@swapkit/toolboxes/cosmos";

async function cosmosWallet() {
  // Get Cosmos toolbox
  const cosmosToolbox = getCosmosToolbox(Chain.THORChain);

  // Create a wallet from a mnemonic phrase
  const phrase = "your twelve word mnemonic phrase here";
  const signer = await cosmosToolbox.getSigner(phrase);
  const address = await cosmosToolbox.getAddressFromMnemonic(phrase);

  return { signer, address }
}
```
