## Installation

Install SwapKit and its dependencies:

<Tabs syncKey="package-manager">
  <TabItem label="bun" icon="bun">
    ```bash bun add @uswap/sdk ```
  </TabItem>
  <TabItem label="pnpm">```bash pnpm add @uswap/sdk ```</TabItem>
  <TabItem label="npm">```bash npm install @uswap/sdk ```</TabItem>
  <TabItem label="yarn">```bash yarn add @uswap/sdk ```</TabItem>
  <TabItem label="expo">```bash expo install @uswap/sdk ```</TabItem>
</Tabs>
