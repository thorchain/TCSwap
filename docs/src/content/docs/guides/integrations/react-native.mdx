This guide will help you integrate SwapKit into your React Native or Expo application.

:::caution[Coming Soon]
React Native support is currently under development and will be available in a future release.

**Current Status:**
- Core SwapKit functionality works in React Native environments
- Wallet connections require additional setup and polyfills
- Full documentation and examples are being prepared

**In the meantime:**
- Consider using SwapKit in a mobile-optimized web app
- Join our [Discord](https://discord.gg/swapkit) for updates on React Native support
- Check back soon for the complete integration guide
:::

## Installation

Install SwapKit and its dependencies:

<Tabs syncKey="package-manager">
  <TabItem label="bun" icon="bun">
    ```bash
    bun add @swapkit/sdk
    ```
  </TabItem>
  <TabItem label="pnpm">
    ```bash
    pnpm add @swapkit/sdk
    ```
  </TabItem>
  <TabItem label="npm">
    ```bash
    npm install @swapkit/sdk
    ```
  </TabItem>
  <TabItem label="yarn">
    ```bash
    yarn add @swapkit/sdk
    ```
  </TabItem>
  <TabItem label="expo">
    ```bash
    expo install @swapkit/sdk
    ```
  </TabItem>
</Tabs>
