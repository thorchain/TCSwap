THORChain provides advanced DeFi features beyond simple swaps. USwap includes full support for these features through the THORChain [plugin](/guides/create-plugin).

## Prerequisites

To use THORChain features, you need:

1. A connected wallet with RUNE
2. USwap initialized with the THORChain plugin (included by default)

```typescript
// @noErrorValidation
import { createUSwap, Chain, FeeOption } from "@tcswap/sdk";

const uSwap = createUSwap();

await uSwap.connectKeystore([Chain.THORChain], "your mnemonic phrase");
```

## Liquidity Operations

THORChain uses an asymmetric liquidity model where you can provide single-sided or dual-sided liquidity.

### Add Liquidity

#### Symmetrical Liquidity (RUNE + Asset)

```typescript
// @noErrorValidation
import { AssetValue } from "@tcswap/sdk";

const runeAmount = AssetValue.from({ asset: "THOR.RUNE", value: 100 });
const ethAmount = AssetValue.from({ asset: "ETH.ETH", value: 0.5 });

const result = await uSwap.thorchain.addLiquidity({
  baseAssetValue: runeAmount,
  assetValue: ethAmount,
  baseAssetAddr: uSwap.getAddress(Chain.THORChain),
  assetAddr: uSwap.getAddress(Chain.Ethereum),
  mode: "sym",
});

console.log("RUNE tx:", result.baseAssetTx);
console.log("ETH tx:", result.assetTx);
```

#### Asymmetrical Liquidity (Single-Sided)

```typescript
// @noErrorValidation

const runeOnly = await uSwap.thorchain.addLiquidity({
  baseAssetValue: runeAmount,
  assetValue: AssetValue.from({ asset: "BTC.BTC", value: 0 }),
  baseAssetAddr: uSwap.getAddress(Chain.THORChain),
  mode: "baseAsset",
});

const btcAmount = AssetValue.from({ asset: "BTC.BTC", value: 0.1 });

const btcOnly = await uSwap.thorchain.addLiquidity({
  baseAssetValue: AssetValue.from({ asset: "THOR.RUNE", value: 0 }),
  assetValue: btcAmount,
  assetAddr: uSwap.getAddress(Chain.Bitcoin),
  mode: "asset",
});
```

#### Create New Liquidity Position

For creating a new liquidity position (bootstrapping a pool):

```typescript
// @noErrorValidation
const result = await uSwap.thorchain.createLiquidity({
  baseAssetValue: runeAmount,
  assetValue: ethAmount,
});

console.log("RUNE tx:", result.baseAssetTx);
console.log("Asset tx:", result.assetTx);
```

#### Add Liquidity Part

For more granular control, you can add liquidity parts separately:

```typescript
// @noErrorValidation
const poolAddress = "BTC.BTC";

await uSwap.thorchain.addLiquidityPart({
  assetValue: btcAmount,
  poolAddress,
  address: uSwap.getAddress(Chain.Bitcoin),
  symmetric: true,
});
```

### Withdraw Liquidity

```typescript
// @noErrorValidation

const withdrawTx = await uSwap.thorchain.withdraw({
  assetValue: AssetValue.from({ asset: "BTC.BTC", value: 0 }),
  percent: 50,
  from: "sym",
  to: "sym",
});

const runeToSym = await uSwap.thorchain.withdraw({
  assetValue: AssetValue.from({ asset: "ETH.ETH", value: 0 }),
  percent: 100,
  from: "baseAsset",
  to: "sym",
});

const toAssetOnly = await uSwap.thorchain.withdraw({
  assetValue: AssetValue.from({ asset: "BTC.BTC", value: 0 }),
  percent: 25,
  from: "sym",
  to: "asset",
});
```

```typescript
// @noErrorValidation
import { USwapApi } from "@tcswap/helpers/api";

const thorAddress = uSwap.getAddress(Chain.THORChain);
```

## THORNames

THORNames are human-readable aliases for wallet addresses across all supported chains.

### Register THORName

```typescript
// @noErrorValidation

const thorname = "myname";
const years = 1;

const details = await USwapApi.thorchainMidgard.getTHORNameDetails(thorname);
if (details.owner) {
  console.log("THORName already taken");
  return;
}

const registrationFee = AssetValue.from({
  asset: "THOR.RUNE",
  value: years + 0.02,
});

const registerTx = await uSwap.thorchain.registerName({
  assetValue: registrationFee,
  name: thorname,
  chain: "THOR",
  address: uSwap.getAddress(Chain.THORChain),
  owner: uSwap.getAddress(Chain.THORChain),
  preferredAsset: "ETH.ETH",
});
```

### Set Preferred Asset

Configure which asset you want to receive when someone sends to your THORName:

```typescript
// @noErrorValidation

const preferredAsset = AssetValue.from({
  asset: "ETH.USDC-0xA0b86991c6218b36c1d19D4a2e9Eb0cE3606eB48",
  value: 0,
});

const setPreferredTx = await uSwap.thorchain.registerPreferredAsset({
  assetValue: AssetValue.from({ chain: Chain.THORChain }),
  name: thorname,
  ownerAddress: uSwap.getAddress(Chain.THORChain),
  payoutAddress: uSwap.getAddress(Chain.Ethereum),
});
```

### Update THORName Aliases

Set addresses for different chains:

```typescript
// @noErrorValidation

const updateBtcTx = await uSwap.thorchain.registerName({
  assetValue: AssetValue.from({ chain: Chain.THORChain }),
  name: thorname,
  chain: "BTC",
  address: uSwap.getAddress(Chain.Bitcoin),
  owner: uSwap.getAddress(Chain.THORChain),
});

const chainMap = {
  [Chain.Ethereum]: "ETH",
  [Chain.Bitcoin]: "BTC",
  [Chain.BinanceSmartChain]: "BSC",
};

for (const [chain, chainId] of Object.entries(chainMap)) {
  await uSwap.thorchain.registerName({
    assetValue: AssetValue.from({ chain: Chain.THORChain }),
    name: thorname,
    chain: chainId,
    address: uSwap.getAddress(chain as Chain),
    owner: uSwap.getAddress(Chain.THORChain),
  });
}
```

### Lookup THORName

```typescript
// @noErrorValidation
const thornameDetails = await USwapApi.thorchainMidgard.getTHORNameDetails(
  "myname"
);

console.log("Owner:", thornameDetails.owner);
console.log("Preferred asset:", thornameDetails.preferredAsset);
console.log("Aliases:", thornameDetails.aliases);

const ethAddress = thornameDetails.aliases.find(
  (a) => a.chain === "ETH"
)?.address;
```

## Node Operations

For node operators, USwap provides bond management functions.

### Bond RUNE

```typescript
// @noErrorValidation
import { MemoType } from "@tcswap/sdk";

const bondAmount = AssetValue.from({ asset: "THOR.RUNE", value: 100000 });
const nodeAddress = "thor1abc...";

const bondTx = await uSwap.thorchain.nodeAction({
  type: MemoType.BOND,
  assetValue: bondAmount,
  address: nodeAddress,
});
```

### Unbond and Leave

```typescript
// @noErrorValidation

const unbondTx = await uSwap.thorchain.nodeAction({
  type: MemoType.UNBOND,
  assetValue: AssetValue.from({ asset: "THOR.RUNE", value: 50000 }),
  address: nodeAddress,
});

const leaveTx = await uSwap.thorchain.nodeAction({
  type: MemoType.LEAVE,
  address: nodeAddress,
});
```

## Direct Deposit

For advanced users, deposit directly to THORChain with custom memos:

```typescript
// @noErrorValidation

const customDeposit = await uSwap.thorchain.deposit({
  assetValue: AssetValue.from({ asset: "BTC.BTC", value: 0.1 }),
  recipient: "",
  memo: "SWAP:ETH.ETH:thor1...",
});

const poolDeposit = await uSwap.thorchain.depositToPool({
  assetValue: AssetValue.from({ asset: "ETH.ETH", value: 1 }),
  memo: "ADD:ETH.ETH:thor1...",
  feeOptionKey: FeeOption.Fast,
});

const protocolDeposit = await uSwap.thorchain.depositToProtocol({
  assetValue: AssetValue.from({ asset: "THOR.RUNE", value: 10 }),
  memo: "BOND:thor1nodeaddress...",
});
```

## Token Approvals

For EVM chains, THORChain requires token approvals before deposits:

```typescript
// @noErrorValidation

const isApproved = await uSwap.thorchain.isAssetValueApproved({
  assetValue: AssetValue.from({
    asset: "ETH.USDC-0xA0b86991c6218b36c1d19D4a2e9Eb0cE3606eB48",
    value: 1000,
  }),
});

if (!isApproved) {
  const approveTx = await uSwap.thorchain.approveAssetValue({
    assetValue: AssetValue.from({
      asset: "ETH.USDC-0xA0b86991c6218b36c1d19D4a2e9Eb0cE3606eB48",
      value: 1000,
    }),
  });

  console.log("Approval tx:", approveTx);
}
```

## Chain Information

Get inbound addresses and gas rates for any supported chain:

```typescript
// @noErrorValidation

const btcInbound = await uSwap.thorchain.getInboundDataByChain(Chain.Bitcoin);

console.log("BTC vault address:", btcInbound.address);
console.log("BTC router:", btcInbound.router);
console.log("Gas rate:", btcInbound.gas_rate);
console.log("Chain halted:", btcInbound.halted);

const chains = [Chain.Ethereum, Chain.Bitcoin, Chain.BinanceSmartChain];
for (const chain of chains) {
  const inbound = await uSwap.thorchain.getInboundDataByChain(chain);
  if (inbound.halted) {
    console.log(`${chain} is currently halted`);
  }
}
```

## Best Practices

### 1. Check Pool Status

Before adding liquidity or making swaps:

```typescript
// @noErrorValidation
const pools = await USwapApi.thorchainMidgard.getPools();
const btcPool = pools.find((p) => p.asset === "BTC.BTC");

if (btcPool?.status !== "Available") {
  console.log("Pool not available");
  return;
}
```

### 2. Calculate Slippage

For large liquidity operations:

```typescript
// @noErrorValidation
const quote = await USwapApi.getSwapQuote({
  sellAsset: "THOR.RUNE",
  sellAmount: "1000000000000",
  buyAsset: "BTC.BTC",
});

const slippage = quote.routes[0].fees.slippage;
console.log(`Slippage: ${slippage}%`);
```

### 3. Monitor Transaction Status

Track THORChain transactions:

```typescript
// @noErrorValidation
const txStatus = await USwapApi.getTrackerDetails({
  txHash: "ABCD...",
  chain: "THOR",
});

if (txStatus.status === "completed") {
  console.log("Transaction successful");
}
```

## API Reference Summary

Here's a complete list of THORChain plugin methods:

### Core Methods

- `deposit()` - Direct deposit with custom parameters
- `depositToPool()` - Deposit to specific pool with memo
- `depositToProtocol()` - Simplified protocol deposits

### Liquidity Methods

- `addLiquidity()` - Add symmetric or asymmetric liquidity
- `addLiquidityPart()` - Add liquidity with granular control
- `createLiquidity()` - Bootstrap new liquidity pools
- `withdraw()` - Withdraw liquidity positions

### THORName Methods

- `registerName()` - Register or update THORNames
- `registerPreferredAsset()` - Set preferred payout asset

### Node Methods

- `nodeAction()` - Bond, unbond, or leave as node operator

### Token Methods (EVM)

- `approveAssetValue()` - Approve tokens for router
- `isAssetValueApproved()` - Check approval status

### Utility Methods

- `getInboundDataByChain()` - Get vault addresses and gas rates
- `swap()` - Execute swaps (used internally by USwap)
